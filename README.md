# mkk

## Project setup
```
npm install
npm install -g ganache-cli
npm install -g truffle
```

### Compiles and hot-reloads for development
##### On linux
```
npm run serve
```
##### On windows
```
truffle.cmd compile
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
